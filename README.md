📦 Gestión de Catálogo de Artículos
📝 Descripción

Aplicación de escritorio para gestionar artículos de un catálogo comercial. Permite listar, buscar, agregar, modificar, eliminar y ver el detalle de productos. Diseñada para adaptarse a distintos tipos de negocios.

⚙️ Funcionalidades
📋 Listado de artículos
🔎 Búsqueda por distintos criterios
➕ Alta de artículos
✏️ Modificación de artículos
❌ Eliminación de artículos
📄 Visualización de detalle
🛠️ Tecnologías
C#
.NET (WinForms)
SQL Server (opcional según implementación)
🚀 Cómo ejecutar
Clonar el repositorio
Abrir la solución en Visual Studio
Compilar el proyecto
Ejecutar la aplicación
📌 Objetivo

Centralizar la gestión de productos para su posterior uso en diferentes plataformas (web, mobile, e-commerce, etc.).
